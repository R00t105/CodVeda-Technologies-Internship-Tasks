## 📂 Project Structure & Tasks Breakdown

The internship tasks were divided into three main progressive levels:

### 🥉 Level 1: Basics & Core Logic
Focuses on fundamental logic, control flow, loops, and basic structures.
- **Task 1: Basic Calculator**
  - A simple console application capable of performing basic arithmetic operations (addition, subtraction, multiplication, division). Handles edge cases like division by zero.
- **Task 2: Simple Number Guessing Game**
  - An interactive game where the system generates a random number, providing real-time feedback (too high/too low) to the user based on their input.
- **Task 3: Factorial Calculation using Recursion**
  - A program that implements recursive methods to calculate factorials, securely handling edge cases such as negative numbers and 0.

### 🥈 Level 2: Intermediate - OOP & File Handling
Focuses strictly on standard Java Object-Oriented principles, File I/O operations, and basic Data Structures.
- **Task 1: Employee Management System**
  - A robust console application implementing **CRUD operations** (Create, Read, Update, Delete) utilizing Java in-memory collections like `ArrayList` and defining robust classes.
- **Task 2: File Handling (Reading and Writing)**
  - A file processing program that demonstrates reading data from text files (e.g., counting words/lines) and safely writing processed data out avoiding `FileNotFoundException`.
- **Task 3: (Additional Intermediate Task)**
  - Advanced operations and practical challenges enhancing Java knowledge.

### 🥇 Level 3: Advanced Concepts
Focuses on building full applications, applying algorithms, or deeper architectural patterns.
- **Task 1: Advanced Application / Project**
- **Task 2: System Implementation**
- **Task 3: Capstone Challenges**

## 🧠 What I Learned & Achieved

This internship profoundly enhanced my software engineering toolkit. Here’s a summary of the key takeaways:

1. **Object-Oriented Programming (OOP):** Solidified principles such as Encapsulation, Inheritance, Polymorphism, and Abstraction to write clean, reusable code.
2. **Java Core Fundamentals:** Deepened my understanding of core APIs, `ArrayList`, exceptions, and memory structures in Java.
3. **Problem Solving & Logic Design:** Enhanced recursive thinking, loops logic, and conditional processing through engaging scenarios (e.g. Number Guessing Game & Factorials).
4. **File Handling & System I/O:** Learned how back-end applications manage persistent data operations interacting with file structures directly.
5. **Effective Exception Handling:** Implemented `try-catch` blocks extensively, effectively anticipating and resolving edge cases (e.g., `ArithmeticException`, `FileNotFoundException`).
6. **Code Structuring:** Divided complex problems into manageable functions and class attributes for high-end readability.

## 🚀 How To Run

To run any of the tasks locally, ensure you have the **Java Development Kit (JDK)** installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Codveda-Tasks.git
   ```
2. **Navigate to the corresponding task directory:**
   ```bash
   cd "Level 1/Task 1"
   ```
3. **Compile the Java file:**
   ```bash
   javac Main.java
   ```
4. **Run the application:**
   ```bash
   java Main
   ```
